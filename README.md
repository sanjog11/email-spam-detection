# Email Spam Detection

A web app to detect whether an email is spam or not using a Logistic Regression model trained on a labeled dataset.

## How to Use
1. Paste your email text into the input box.
2. Click "Predict".
3. The model will tell you if it's spam or not spam.

## Built With
- Python
- Scikit-learn
- Streamlit
- TF-IDF
